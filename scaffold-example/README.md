# scaffold-example

A basic plugin scaffold example for Claude Code.

## Installation

```bash
claude plugin install /path/to/scaffold-example
```

## What's included

This scaffold provides the minimum files needed for a Claude Code plugin:

- Plugin metadata files
- Basic documentation

## Extending

You can add:

- `commands/` - Custom slash commands
- `skills/` - Skills with YAML frontmatter
- `agents/` - Custom subagents
- `hooks/` - Pre/post tool hooks
- `output-styles/` - Custom output styles
- `settings.json` - Plugin-specific settings
